# Jeopardy
A responsive Jeopardy-style quiz web app with category selection, point-based scoring, and dynamic question rendering using modern web technologies.


# 🎯 Jeopardy Multiplayer Quiz Game

An interactive multiplayer Jeopardy-style quiz game built with a modern full-stack architecture. Players can join game rooms, compete in real-time, and test their knowledge across multiple categories.

---

## 🚀 Tech Stack

### 🖥️ Frontend
- React.js
- Tailwind CSS / CSS Modules
- Axios / Fetch API

### ⚙️ Backend
- Spring Boot (Java)
- REST APIs
- WebSockets (for real-time gameplay)

### 🗄️ Database
- PostgreSQL

---

## ✨ Features

- 🎮 Multiplayer game rooms
- 📚 Category-based questions (Jeopardy board)
- ⏱️ Real-time question answering
- 💯 Dynamic score tracking
- 🔄 Turn-based gameplay
- 🔐 User authentication (optional)
- 📊 Leaderboard (optional)

---

## 🏗️ System Architecture

Frontend (React) ↔ Backend (Spring Boot REST + WebSocket) ↔ PostgreSQL
